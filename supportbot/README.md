# SupportBot Pro

Многофункциональный модульный бот для саппорта в Telegram с админ-меню, фичами на включение, мощным FAQ, AI, рассылками, push и кроссплатформенностью.

## Возможности

- Контекстная поддержка (хранит недавние вопросы пользователя для анализа)
- Модули: файлы, мультиязык, персональные автоответы, QR-сканер, push-уведомления, AI-помощник, обратная связь и рефералы, кроссплатформенность, рассылки, умный FAQ.
- Всё включается/выключается через команду /admin_toggle <module> on|off (только для admin).
- Простая интеграция, гибкая архитектура, готово к деплою в cloud.
